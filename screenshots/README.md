# Screenshots

This folder contains screenshots of the Vaarta chat application.

## How to Add Screenshots

1. **Take screenshots** of your app on different devices/screens
2. **Save them** in this `screenshots/` folder
3. **Name them descriptively** (e.g., `login_screen.png`, `chat_conversation.png`)
4. **Update the README.md** to reference the new screenshots
5. **Commit and push** the changes to GitHub

## Recommended Screenshot Categories

- **Authentication**: Login, signup, password reset screens
- **Main Interface**: Chat list, conversation view, navigation
- **Media Features**: Image/video sharing, camera integration
- **Settings**: Profile, preferences, theme selection
- **Special Features**: Contacts, status updates, emoji picker

## Image Guidelines

- **Format**: PNG or JPG
- **Resolution**: High quality but optimized for web (under 2MB per image)
- **Naming**: Use lowercase with underscores (e.g., `chat_list.png`)
- **Size**: Aim for consistent dimensions across similar screenshots

## Example Usage in README

```markdown
<img src="screenshots/login_screen.png" alt="Login Screen" width="300"/>
```

This will display the screenshot in the README with a width of 300px.
